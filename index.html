2. CSS Początki to używanie podstawowych "style" na pojedynczych elementach w sekcji "head".

    Póki co :
  • - < color>
  • - < background>

Komenda!

Zalinkowanie arkusza styli css (który piszemy w osobnej karcie) w arkuszu html.

    < link href="stylecss" rel="stylesheet" Type="text/css">

3. Terminal Windows

Podstawy w poruszaniu się po terminalu.
  • -cd - wyświetlenie aktualnej ścieżki
  • -cd "nazwa folderu" - idź do katalogu
  • -dir - zawartość katalogu
  • -cd .. - idź do katalogu wyżej
  • -mk dir "nazwa katalogu" - stwórz katalog
  • -code . - otwórz program visual studio
